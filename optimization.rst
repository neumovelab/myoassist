Optimization Framework
=======================

MyoAssist uses CMA-ES (Covariance Matrix Adaptation Evolution Strategy) for optimizing neuromuscular reflex controller parameters.

Key Features
-----------

- **CMA-ES Optimization**: Efficient parameter optimization
- **Multi-stage Cost Functions**: Configurable objectives
- **Parameter Bounds**: Feasible parameter ranges
- **Multi-objective Support**: Balance multiple objectives

For detailed optimization procedures, see :doc:`running_optimizations`.

For cost function details, see :doc:`understanding_cost`.
