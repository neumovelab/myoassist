Welcome to MyoAssist's documentation!¶

MyoAssist is a collection of neuromuscular reflex controller optimization environments and tasks simulated with the MuJoCo physics engine and wrapped in the OpenAI Gym API to enable the application of Machine Learning to bio-mechanic control problems.

Check our github repository for more technical details.

Our paper can be found at: https://arxiv.org/abs/2205.13600

Advanced users are invited to familiarize themselves with the basics of the OpenAI Gym API and review the basic principle of Reinforcement Learning to make the most out of MyoAssist features and functionalities

Note

This project is under active development.
