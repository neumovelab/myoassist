MyoAssist

A Python-based framework for optimizing neuromuscular reflex controllers for human locomotion, with support for exoskeleton assistance.

Get Started View on GitHub

Key Features

Modular Architecture

Extensible framework for neuromuscular and exoskeleton control with support for multiple musculoskeletal models.

CMA-ES Optimization

Advanced optimization algorithms for controller parameters with configurable multi-stage cost functions.

Exoskeleton Support

Support for multiple exoskeleton hardware platforms including Dephy, HMEDI, Humotech, and OSL.
